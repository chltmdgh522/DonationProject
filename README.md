### ⚙️ 개발 환경
- `Java 17`
- **Framework** : Springboot(3.x)
- **Database** : MySql
- **ORM** : JdbcTemplate, JPA
- **IDE** : IntellJ

## 📌 주요 기능
#### 로그인/회원가입 -
- DB값 검증
- 비밀번호 찾기 및 변경 
- 로그인 시 쿠키(Cookie) 및 세션(Session) 생성

#### 마이 페이지 
- 프로필, 이름, 소개 변경

#### 게시판 
- CRUD 제공
- 댓글 기능 제공(CRUD)
  
#### 후원 및 포인트 
- 후원 목표액 설정
- 회원가입/로그인/게시물 작성 할 때마다 포인트 증가
- 후원 목표액 달성하면 후원하기 버튼 안뜸
  
#### 관리자 페이지 
- 회원관리, 댓글관리, 게시물관리
